Information Storage, modular programming and data structures

Information Storage, modular programming and data structures

Periods: 30

This unit focuses on the advanced features of programming. For example students learn about data storage and manipulation. In addition modular programming is introduced. Data structures such as arrays are taught. Common algorithms like the Bubble Sort are examined. Students are expected to incorporate as many of these more complex programming features in their final summative project.
